# Lu Doces - Site Responsivo

## 📋 Descrição
Site moderno e totalmente responsivo para a confeitaria Lu Doces, desenvolvido com HTML5, CSS3 e JavaScript vanilla.

## ✨ Características Principais

### 🎨 Design Moderno
- Design limpo e profissional
- Paleta de cores atrativa (rosa/vermelho)
- Tipografia elegante com fontes Google (Dancing Script + Poppins)
- Ícones Font Awesome para melhor experiência visual

### 📱 Totalmente Responsivo
- Layout adaptável para todos os dispositivos
- Menu hambúrguer para mobile
- Grid system flexível
- Breakpoints otimizados:
  - Desktop: > 768px
  - Tablet: 768px - 480px
  - Mobile: < 480px
  - Mobile pequeno: < 360px

### 🚀 Funcionalidades
- **Navegação suave** entre seções
- **Menu mobile** interativo
- **Animações** de scroll
- **Formulário de contato** funcional
- **Botão WhatsApp flutuante** para contato direto
- **Efeitos hover** nos elementos
- **Navbar com blur** e transparência
- **Scroll automático** para seções

### 📂 Estrutura do Site
1. **Header/Navbar** - Navegação fixa com logo e menu
2. **Hero Section** - Seção principal com call-to-action
3. **Sobre** - Informações sobre a empresa
4. **Produtos** - Cards dos produtos oferecidos
5. **Contato** - Formulário e informações de contato
6. **Footer** - Links sociais e informações adicionais

## 🛠️ Tecnologias Utilizadas
- **HTML5** - Estrutura semântica
- **CSS3** - Estilos modernos com Flexbox e Grid
- **JavaScript** - Interatividade e animações
- **Font Awesome** - Ícones
- **Google Fonts** - Tipografia

1. Abra o arquivo `index.html` em qualquer navegador moderno

   - Tablet
   - Smartphone
   - Qualquer dispositivo móvel

## 📱 Teste de Responsividade

Para testar a responsividade:
1. Abra o site no Chrome/Firefox
2. Pressione F12 para abrir as ferramentas de desenvolvedor
3. Clique no ícone de dispositivo móvel
4. Teste diferentes resoluções
